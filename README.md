# Gestionnaire de stocks

Un simple gestionnaire Stock en ligne de commande en C

---

# Comment installer ?

```bash
git clone https://github.com/Starland9/gestion-de-stocks-c.git
cd gestion-de-stocks-c
```

```bash
make
```

```bash
make run
```

## Windows

```bash
./bin/stocky.exe
```

## Linux

```bash
./bin/stocky
```
